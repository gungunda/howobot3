Тестовый контур для проекта «Лёшин планировщик»
--------------------------------------------------
1) Скопируй папку tests/ в корень репозитория (рядом с index.html).
2) Открой в браузере tests/test.html (или подними локальный сервер).
3) Нажми кнопку Run tests — увидишь отчёт по тестам.

Важно:
- В test.html настроен import map: /js/app.js -> /tests/mocks/app.test.js
- Тесты ожидают, что реальный код usecases лежит по путям /js/usecases/*.js
- Если у тебя другие путя — поправь импорты в файлах tests/units/*.test.js
